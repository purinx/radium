# Radium HTML Specification

RadiumがサポートするHTML要素・動作の仕様を定義する。

---

## 1. パーサー

- タグ名は大文字・小文字を区別しない。
- テキストノード内の連続する空白文字（スペース・タブ・改行）は単一のスペースに折り畳む。
- 先頭・末尾の空白はトリミングする。
- `<!DOCTYPE>` 宣言はサポートし、無視する。
- 閉じタグが対応する開きタグと一致しない場合、最も近い一致タグまで自動クローズする（エラー回復）。
- 属性は構文上受け付けるが、すべて無視する。

---

## 2. サポート要素

### 2.1 スキップ要素

以下の要素とその子孫は表示されない。

- `head`, `title`, `script`, `style`, `meta`, `link`

### 2.2 コンテナ要素

以下の要素はスタイルを変更せず、子要素をそのまま描画する。

- `html`, `body`, `div`, `section`, `article`, `main`, `header`, `footer`

未知のタグも同様にコンテナとして扱う。

### 2.3 見出し

| タグ | フォントサイズ | 上マージン | 下マージン |
|------|-------------:|----------:|----------:|
| `h1` | 32px | 24px | 16px |
| `h2` | 24px | 20px | 12px |
| `h3` | 20px | 16px | 8px |

- すべての見出しは太字で表示する。
- `h4`〜`h6` は未対応（コンテナとして処理）。

### 2.4 段落

| タグ | 下マージン |
|------|----------:|
| `p`  | 16px |

### 2.5 リスト

| タグ | 説明 |
|------|------|
| `ul` | 箇条書きリスト |
| `ol` | 番号付きリスト |
| `li` | リストアイテム |

- リスト全体の上下に8pxのマージンを追加する。
- ネストは任意の深さをサポートする。

#### 箇条書きマーカー（ネスト深度別）

| 深度 | マーカー |
|-----:|--------|
| 1 | `•` |
| 2 | `◦` |
| 3以上 | `▪` |

#### 番号付きリストのマーカー形式

`1.`, `2.`, `3.` … （1始まりの連番、ドット区切り）

### 2.6 インライン要素

| タグ | 表示 |
|------|------|
| `strong` | 太字 |
| `em` | 斜体 |
| `a` | 青色（`#0000EE`）・アンダーライン |
| `span` | スタイル変更なし |

### 2.7 Void要素

| タグ | 動作 |
|------|------|
| `br` | 改行 |
| `hr` | 水平線 |
| `img` | ローカル画像を表示 |

- 自己閉鎖構文（`/>`）もサポートする。

### 2.8 画像（`img`）

- `src` 属性にサイトディレクトリからの相対パスを指定する。
- ネットワークURLは非対応。
- 対応フォーマット: PNG, JPEG
- コンテンツ幅を超える場合、縦横比を維持してスケールダウンする。
- `src` 以外の属性（`width`, `height`, `alt` 等）はすべて無視する。

---

## 3. 非対応事項

- CSS（外部・インライン・`<style>` タグすべて）
- `<h4>` 〜 `<h6>`
- フォームコントロール（`<input>`, `<button>`, `<select>` 等）
- テーブル（`<table>`, `<tr>`, `<td>` 等）
- HTMLエンティティ（`&amp;`, `&lt;` 等）のデコード
- 属性値の参照（`class`, `id`, `href`, `data-*` 等）
- JavaScript（`<script>` は無視）
- テキストの折り返し・ワードラップ
